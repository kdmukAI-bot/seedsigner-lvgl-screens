# screen_runner

Interactive desktop runner for SeedSigner C-module screens.

Status: scaffold only (implementation in progress).

Planned responsibilities:
- load scenarios from `../scenarios.json`
- provide live scenario selector UI (`screen / variation`)
- map keyboard input (arrows, Enter, 1/2/3)
- focus handoff into rendered screen after scenario load

See also:
- `../../docs/SCREEN_RUNNER.md`
